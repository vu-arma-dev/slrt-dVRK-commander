A, B, C, D: four screw head on the force sensor.
These four points can determine the (X-Y) plane of the force sensor.
Details of these four screw heads are case by case, for gamma force sensor, it is shown
https://github.com/vu-arma-dev/slrt-dVRK-commander/wiki/Force-controlled-scan-xPC

center - a point right above the center of the force sensor. (can project this point on  XY plane)
pin - a point right above the x+ direction. (can project this point on XY plane.)
